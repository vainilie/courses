# Proceso de desarrollo de software

- **Tipo de actividad:** Colaborativa
- **Lista de Cotejo:** [[CFDS-U7-LC-ProcesoDesarrollo.pdf]]

**Instrucciones:**

1. **Formen** equipos de **dos** personas para trabajar de manera colaborativa 2. **Elijan** un producto de software, puede ser uno nuevo o uno ya existente
2. **Comenten** qué acciones se realizan en cada fase del desarrollo de software específicamente para el proyecto que seleccionaron
3. **Elaboren** una tabla o esquema que incluya las fases del desarrollo de software, con la siguiente información: 
	- Fase de planeación: definan el alcance del proyecto y realicen un cronograma de trabajo con al menos tres tareas
	- Fase de análisis: expliquen al menos dos requerimientos
	- Fase de diseño: propongan una metodología de trabajo 
	- Fase de desarrollo: describan al menos dos procedimientos
	- Fase de pruebas: expliquen al menos dos pruebas unitarias
	- Fase de mantenimiento: planteen al menos una estrategía de respaldos 
4. **Reflexionen** acerca de la importancia de cada fase y **realicen** una conclusión, que deberá incluirse en el reporte, en la que respondan preguntas como: ¿Qué pasaría si no se ejecuta alguna? ¿Cuál es la fase a la cual, se le debe asignar mayor tiempo?
5. **Elaboren** un reporte en equipo en donde integren el desarrollo y conclusión de la actividad
6. **Guarden** su actividad en formato .pdf y **entreguen** el documento en el espacio correspondiente, indicando los nombres de los participantes

Antes de enviar la actividad, revisa la calidad de ésta, con la **lista de cotejo.**
