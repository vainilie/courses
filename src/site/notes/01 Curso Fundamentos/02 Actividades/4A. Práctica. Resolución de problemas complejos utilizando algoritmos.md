# Resolución de problemas complejos utilizando algoritmos

- **Tipo de actividad:** individual
- **Manual:** [[CFDS-U4-MP-ResolucionProblemas.pdf]]
 - **Lista de Cotejo:** [[CFDS-U4-LC-ResolucionProblemas.pdf]]

**Instrucciones:**

1. **Descarga** el manual y **sigue** los pasos para realizar tu práctica
2. **Elabora** un reporte de práctica que contenga introducción, desarrollo y conclusiones e **integra** evidencia gráfica de los pasos que seguiste con base en el manual
3. **Guarda** tu reporte en formato .pdf y **entrega** el documento junto con los archivos .psc con el código de tus algoritmos en el espacio correspondiente
4. Antes de enviar tu actividad, revisa la calidad de esta con la **lista de cotejo.**
